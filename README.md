# DrugResistanceEngine

Analyze drug resistance mutations including thermodynamic stability changes, fitness costs, and clinical variant mapping.

## Features

- Resistance mutation delta-delta-G stability change prediction
- Fitness cost estimation for resistance variants
- Clinical mutation database cross-referencing
- Escape variant trajectory modeling
- Multi-drug resistance mechanism classification

## Results

5 drugs × 200 mutations; 179 clinical (89.5%); Mean delta-delta-G=2.43 kcal/mol; Mean fitness cost=0.109

## Usage

```bash
pip install numpy scipy matplotlib
python drug_resistance_engine.py
```
